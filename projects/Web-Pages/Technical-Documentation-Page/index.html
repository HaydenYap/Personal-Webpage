Hello World

Here is an example of the famous "Hello World" program written in JavaScript

function helloWorld(name){ alert("Hello World by " + name); } helloWorld("World");

This will output the line "Hello World" along with some extra stuff a more normal version would be:

function helloWorld(name){ alert("Hello " + name); } helloWorld("World");
Variables

What are variables in Java Script?

Variable must start with a letter, underscore or dollar sign followed by letters from A/a to Z/z or digits 0-9. Pretty standard stuff.

Declaring Variables

there are three ways to declare a variable in JS.

var x = 42.

This can be used to declare local and global variables

x = 42.

always declares a global variable and generates a JS waring (so don't use this)

let y=13.

can be used to declare a block scope local variable.

Data Types

there are 7 data types, an Object and 6 primitive data types

  • Boolean - true and false
  • null - special keyword for denoting a "null" value, null is not the same as NULL,Null,... etc
  • undefined - top-level property whose value is undefined
  • Number - any REAL number
  • string - a sequence of characters
  • Symbol - a data type who instances are unique and immutable
